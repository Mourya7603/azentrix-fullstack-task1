# 💰 Personal Budget Tracker

A clean, professional web application for tracking daily income and expenses. Built with vanilla JavaScript, this tool helps individuals and small businesses manage their finances without the complexity of traditional accounting software.

## ✨ Features

- **Complete CRUD Operations** - Add, edit, and delete income/expense entries
- **Interactive Dashboard** - Visual summary with real-time updates
- **Category-Based Charts** - Bar chart showing net income/expense per category
- **Persistent Data** - Automatically saves to localStorage (no server required)
- **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- **Professional UI** - Clean, modern interface with intuitive navigation

## 📋 How It Works

### Adding Entries
1. Fill in the form fields:
   - **Description**: Name of the income/expense
   - **Amount**: Dollar amount (must be > 0)
   - **Category**: Select from predefined categories
   - **Type**: Choose between Income or Expense
   - **Date**: Select the transaction date
2. Click the "Add" button to save

### Managing Entries
- **Edit**: Click the pencil icon (✏️) on any entry to modify it
- **Delete**: Click the trash icon (🗑️) to remove an entry
- **Cancel Edit**: Click the "Cancel" button while in edit mode

### Dashboard Overview
- **Income Card**: Total monthly income
- **Expense Card**: Total monthly expenses
- **Net Balance**: Income minus expenses
- **Category Chart**: Visual breakdown of net spending by category

## 🛠️ Technology Stack

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with Flexbox/Grid
- **Vanilla JavaScript** - No frameworks required
- **Chart.js 3** - For interactive charts
- **Font Awesome 6** - Professional icons
- **localStorage** - Client-side data persistence
